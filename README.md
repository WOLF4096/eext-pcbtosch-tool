# 功能如标题所示
### 这是一个PCB转原理图工具，支持千量级元件数量，支持多部件元件，支持生成对应引脚导线，生成源码速度极快
# 支持嘉立创EDA V2.2.x 和 V3.x 版本
实测 **在 V2.2.45 客户端稳定性较好**，当写入的数据量过大时，V3.2.91版本会出现白屏的情况，可以导出工程文件后手动导入

# 使用方法
### 打开对应板子的原理图，打开插件 ⇒ 同意《免责声明》 ⇒ 加载数据 ⇒ 构建源码 ⇒ 写入文档 / 导出文件(V2.2格式)

# 指定符号
### 在指定符号输入框粘贴对应符号的名称即可（必须是当前列表里的符号）
**也可以输入预设符号：电阻、电感、无极电容**

# 注意事项
## 符号异常处理
  - 在工程库里搜索对应器件，右键编辑器件，选定一个原理图符号
## 在V3版本覆写文档时，需要手动清空画布再写入
## 当写入的数据量过大时，V3版本会出现白屏的情况，可以导出工程文件后手动导入

> 进阶说明：本工具主要依赖工程文件（.epro）解析，理论上只要提供正确的工程文件，稍作修改代码即可在不打开EDA软件的情况下进行PCB转原理图。欢迎有经验的用户自行探索。
